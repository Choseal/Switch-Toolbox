gx2dec.exe - third-party component NOTICE
=========================================

gx2dec.exe is a standalone command-line tool that decompiles Wii U GX2 (AMD
"Latte" R7xx) vertex and pixel shader microcode to GLSL. Its decompiler core is
derived from Cemu (the Wii U emulator), licensed under the Mozilla Public
License 2.0 (MPL-2.0).

  Upstream:  Cemu (https://github.com/cemu-project/Cemu), commit 079a4af
  License:   Mozilla Public License 2.0 (https://www.mozilla.org/MPL/2.0/)

Switch-Toolbox runs this binary as a separate process (see
File_Format_Library/FileFormats/Effects/Gx2ShaderDecompiler.cs); the MPL-covered
code is not linked into the toolbox.

Source: the CLI driver and the build script that links the Cemu decompiler sources
are published under File_Format_Library/gx2dec_src/ (see its README.md), satisfying
MPL-2.0's requirement that the source of the covered files be available.
